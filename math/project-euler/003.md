## The problem

The prime factors of 13195 are 5, 7, 13 and 29.

What is the largest prime factor of the number 600851475143 ?


## The solution

```python
from math import sqrt, ceil

def prime_factors(n):
    p, num, s = 2, n, set()
    while p <= ceil(sqrt(n)):
        while num % p == 0:
            s.add(p)
            num //= p
        p += 1
    if num > 2:
        s.add(num)
    return max(s)


t = int(input().strip())
for a0 in range(t):
    n = int(input().strip())
    print(prime_factors(n))
```
